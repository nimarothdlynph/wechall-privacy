Suppression de compte - WeChall

Dernière mise à jour : 2 juillet 2026

Cette page explique comment demander la suppression de votre compte WeChall et des données associées.

1. Comment demander la suppression de votre compte

Pour demander la suppression de votre compte WeChall, envoyez un e-mail à l’adresse suivante :

E-mail : th.longin@gmail.com

Objet conseillé : Suppression de compte WeChall

Dans votre message, veuillez indiquer :

  • l’adresse e-mail utilisée pour votre compte WeChall ;
  • votre nom d’utilisateur si vous le connaissez ;
  • que vous souhaitez supprimer votre compte et les données associées.

2. Données supprimées

Lorsque votre demande est traitée, les données associées à votre compte peuvent être supprimées, notamment :

  • votre compte utilisateur ;
  • votre nom d’utilisateur ;
  • votre adresse e-mail associée au compte ;
  • votre photo de profil ;
  • vos données de profil ;
  • vos contenus liés aux défis, lorsque leur suppression est techniquement possible ;
  • vos messages, commentaires ou interactions associés à votre compte, lorsque leur suppression est techniquement possible ;
  • vos participations à des groupes ou communautés.

3. Données pouvant être conservées temporairement

Certaines données peuvent être conservées temporairement si cela est nécessaire pour des raisons de sécurité, de prévention des abus, de résolution de problèmes techniques ou d’obligations légales.

Ces données sont conservées uniquement pendant la durée nécessaire à ces finalités.

4. Délai de traitement

Les demandes de suppression sont traitées dans un délai raisonnable après réception de la demande. Une confirmation pourra vous être envoyée lorsque la suppression aura été effectuée.

5. Suppression depuis l’application

Lorsque cette fonctionnalité est disponible dans l’application, vous pouvez également initier une demande de suppression de compte depuis les paramètres de WeChall.

6. Contact

Pour toute question concernant la suppression de compte ou les données personnelles, vous pouvez nous contacter à :

th.longin@gmail.com
